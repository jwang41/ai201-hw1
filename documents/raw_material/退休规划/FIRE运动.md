## 核心概念

**FIRE = Financial Independence, Retire Early**——通过极高储蓄率（通常 50-70%）和投资增长，在传统退休年龄（65）前达到"不工作也能活"的财务状态。

核心公式：
```
FIRE 数字 = 年支出 × 25（基于 4% 规则）
```

随着运动发展，分化出多种"FIRE 流派"。

## 五大流派对比

| 流派 | 年支出目标 | FIRE 数字 | 典型路径 |
|------|-----------|----------|---------|
| **Lean FIRE** | < $40,000 | < $1M | 极简生活 + 地理套利 |
| **Regular FIRE** | $40k-100k | $1-2.5M | 中产阶级标准 FIRE |
| **Fat FIRE** | > $100,000 | > $2.5M | 高收入 + 高支出 |
| **Coast FIRE** | 任意 | 达到足够让复利接管的"Coast Number" | 提前存够，后续只覆盖生活费 |
| **Barista FIRE** | 60-80% 自给 | 对应"半退休" | 组合 + 兼职补贴（含健保） |

### Lean FIRE（精简版）

- **哲学**：极简生活，**退休数字 < $1M**
- **典型年支出**：$20-40k
- **策略**：搬到低成本地区（美国中西部、国际地理套利）、自煮、无车、小房
- **风险**：通胀侵蚀、医疗意外、一旦不得不"回来工作"技能可能过期
- **谁适合**：独身、节俭主义者、愿意大幅降低生活标准

### Fat FIRE（富豪版）

- **哲学**：**退休但生活品质不降**，甚至更高
- **典型年支出**：$100-300k+
- **退休数字**：$2.5M-$10M+
- **策略**：高收入职业（科技、医疗、金融）+ 高储蓄率 + 激进投资
- **风险**：**需要非常高的收入**（年 $300k+ 才能快速积累）、税务效率要求高
- **谁适合**：湾区 / 纽约 科技高管、双高收入夫妻

### Coast FIRE（滑行版）

- **哲学**：**提前存够，剩下交给复利**
- **Coast Number**：达到这个数字后，**不再需要存钱**，让它自然增长到传统退休年龄
- **公式**：
  ```
  Coast Number = 目标 FIRE 数字 / (1 + r)^n
  其中 r = 预期年化回报（通常 5-7% 实际回报），n = 距离目标退休年数
  ```
- **示例**：40 岁，65 岁退休要 $2M，假设 6% 实际回报：
  ```
  Coast Number = $2,000,000 / (1.06)^25 = $466,000
  ```
  → 40 岁存够 $466k 后，就可以"躺平"，只需覆盖日常支出
- **谁适合**：高储蓄前期 → 想降低工作强度（带娃、创业）、但不完全退休

### Barista FIRE（咖啡师版）

- **哲学**：**半退休**——组合覆盖大部分开销，兼职工作补足 + 健保
- **命名由来**：星巴克兼职福利（兼职 20 小时 +）含健保，是最典型的"Barista FIRE"载体
- **组合覆盖**：60-80% 基础开销
- **兼职覆盖**：20-40% + 社交 + 健保
- **示例**：FIRE 需 $1.5M，但有 $800k + Barista 兼职年入 $30k + ACA 补贴 → 可以停下来
- **美国独有优势**：健保绑定就业，Barista FIRE 专门为此设计
- **谁适合**：组合未到 Full FIRE，但已到 70-80%、想脱离主业、享受部分自由

### Regular FIRE / Traditional FIRE

- 中产标准 FIRE，年支出 $40-100k，组合 $1-2.5M
- 大多数 FIRE 社区讨论的"默认"场景

## 其他衍生流派

### Chubby FIRE
Fat 和 Regular 之间，年支出 $80-150k，组合 $2-4M。

### Dynamic FIRE
Barista + Coast 的混合 —— 根据市场动态切换模式。

## 关键数字：储蓄率决定时间

| 储蓄率 | 到 FIRE 所需年数（假设 5% 实际回报）|
|-------|----------------------------------|
| 10% | 51 年 |
| 25% | 32 年 |
| 50% | 17 年 |
| 65% | 11 年 |
| 75% | 7 年 |

**50% 储蓄率是 FIRE 的"分水岭"**。

## 共同关键假设（可能风险）

- ✅ 4% 规则（见 [[4%规则]]）—— 但早退休期更长，可能需 3-3.5%
- ✅ 持续股市回报 —— 但 [[Sequence-of-Returns-Risk]] 风险翻倍
- ✅ 医疗保险可行 —— 美国 Medicare 前需 ACA 或 Barista 路径
- ✅ 通胀 2-3% —— 实际可能更高
- ⚠️ 低估生活变化成本（结婚、生子、父母医疗）

## FIRE 与账户位置（美国）

### 提取顺序

```
退休前 (工作期):
  - 401K + Backdoor Roth + Mega Backdoor Roth (拉满)
  - Taxable brokerage（弹性）
  - HSA（三重税优）

早退休阶段 (59.5 前):
  - Taxable brokerage (随便取)
  - Roth IRA 本金（随时可取，不含增值）
  - Rule 72(t) SEPP（从 401K/IRA 提取无罚款）
  - Roth Conversion Ladder（5 年前转换，今年取）

59.5+ 后:
  - Traditional 401K/IRA
  - Roth IRA 增值（免税）
  - HSA（保留到最后）
```

## 加州高税区 FIRE 挑战

- **高生活成本**：$60k 在奥克兰 ≠ $60k 在俄亥俄
- **高边际税**：加州 9.3% → Coast Number 需上调 10-15%
- **房产因素**：是买 vs 租 FIRE 策略核心差异
- **Prop 13**：长期持有房产锁定税基，是加州独有 FIRE 工具
