## Part 1: 教育规划

### 2026 美国大学费用估算

| 类型 | 年度总费用（学费+食宿）| 4 年合计 |
|------|---------------------|---------|
| 公立州内 | $25,000-$35,000 | $100-140k |
| 公立州外 | $45,000-$55,000 | $180-220k |
| **私立**（Top 50）| **$80,000-$95,000** | **$320-380k** |
| 社区学院 | $10,000-$15,000 | $40-60k（2 年 → 转学）|

### 学费通胀率

历史平均 **~5-6%/年**（远高于一般通胀 2-3%）：
- $80k/年（2026）→ 18 年后（2044）→ 约 **$193k/年**
- **如果孩子刚出生，4 年私立大学总额可能 $800k+**

### FAFSA 策略（简化）

**Expected Family Contribution (EFC) / Student Aid Index (SAI)**：

| 资产类型 | FAFSA 影响率 |
|---------|------------|
| 学生资产（UTMA） | **20%** → 最糟 |
| 家长资产（529、Taxable）| **5.64%** → 中等 |
| 退休账户（401K/IRA/Roth）| **0%** → 最优 |
| 自住房权益 | **0%**（FAFSA 不算）|
| **祖父母 529** | **0%**（2024 FAFSA 简化后）⭐ |

**策略**：
1. **最大化退休账户** → 降低 FAFSA 可见资产
2. **祖父母开 529** → 对 FAFSA 零影响
3. **避免学生名下大额资产**（UTMA 等）
4. **高 Aid 年份**（大一/大二）收入尽量低（如 gap year）
5. **CSS Profile 学校**（私立）会看更多资产 → 另一套规则

### 中美教育路径对比

| | 美国大学 | 中国大学 |
|---|--------|---------|
| 费用 | $25-95k/年 | ¥5-30k/年 |
| 质量 | 全球顶尖（Top 100 多数） | 快速提升（清北复交）|
| 就业 | 美国就业好 | 中国就业好 |
| 身份 | 维持美国身份 | 可能影响绿卡/签证 |
| 语言 | 英语 | 中文 |

**跨境家庭选择**：视孩子未来方向（留美 vs 回国 vs 双边）。

## Part 2: 风险管理框架

### 四种风险应对

| 策略 | 适用场景 | 工具 |
|------|---------|------|
| **Avoid**（规避）| 完全不参与 | 不买投机股 |
| **Reduce**（减少）| 降低概率/影响 | 安全带、运动 |
| **Transfer**（转移）| 交给保险公司 | 购买保险 |
| **Accept**（承受）| 小额可自负 | 紧急基金 |

### 职业风险

**失业/减薪**（科技行业 2023-2025 大裁员经验）：

| 风险 | 应对 |
|------|------|
| 裁员 | 6-12 月紧急基金 + Skill 更新 |
| 行业衰退 | 副业/被动收入 + 可转移技能 |
| 工资停滞 | 跳槽策略 + 谈判技能 |
| 创业失败 | 有限资金投入 + 配偶保留就业 |

### 健康风险

| 风险 | 应对 |
|------|------|
| 重疾（癌症、心脏） | 健康保险（ACA/雇主）+ HSA + 紧急基金 |
| 失能 | **Own-Occupation Disability Insurance**（见 [[保险规划总纲]]）|
| 长期护理 | LTC Insurance 或自保（>$2M 净资产）|

**关键数据**：
- 25% 的 20 岁人会在退休前失能
- 70% 的 65+ 人需要某种长护
- 一次癌症治疗 $100-500k

### 法律风险

| 风险 | 应对 |
|------|------|
| 车祸诉讼 | Auto Liability $250k+ + **Umbrella $1-2M** |
| 房客受伤 | Landlord Insurance + Umbrella |
| 专业过失 | E&O / Professional Liability |
| 个人诉讼 | Umbrella + Asset Protection Trust |

**Umbrella Insurance 极划算**：$200-500/年 买 $1-2M 保护。

### 通胀风险

| 工具 | 保护度 |
|------|-------|
| **TIPS** | ✅ 最直接（CPI 调整） |
| **I Bonds** | ✅ 最安全（固定 + CPI） |
| **股票**（长期） | ✅ 公司利润随通胀涨 |
| **房产** | ✅ 租金/房价随通胀 |
| **黄金** | ✅ 传统避险 |
| 现金/HYSA | ❌ 购买力缩水 |
| 长期固定利率债券 | ❌ 通胀侵蚀实际回报 |

### 身份/税务风险（跨境家庭特有）

| 风险 | 应对 |
|------|------|
| 绿卡过期/丢失 | 及时续期，不超 6 个月境外 |
| 意外成为中国税务居民 | 每年离境 >30 天（见 [[退休回国税务]]）|
| FBAR/8938 漏报 | 每年清单化申报流程 |
| PFIC 陷阱 | 不持有中国基金/理财 |
| Exit Tax 意外触发 | 8 年窗口管理（见 [[Exit-Tax]]）|

## 年度风险审计清单

✅ 紧急基金 ≥ 6 个月？
✅ 保险覆盖充分？（Health + Disability + Life + Umbrella）
✅ 受益人和 POA 更新？
✅ 遗嘱/Trust 还有效？
✅ FBAR/8938 按时提交？
✅ 数字遗产清单更新？
✅ 投资集中度 < 20%？
✅ 债务利率合理？
