## 核心概念

**Power of Attorney (POA) = 你授权他人代你做决定的法律文件**

**关键区别**：
- **Financial POA**（财务委托书）：代你处理财产、账户、交易
- **Healthcare POA**（医疗委托书）：代你做医疗决定（如果你不能）

⚠️ **两者完全独立** —— 一份不能替代另一份，加州法律下**都需要**。

## Durable（持久）vs Non-Durable

**Durable POA**（持久委托书）= **失能后继续有效**

**Non-Durable POA** = 你失能时**立即失效**（比如银行临时授权）

**遗产规划用途必须选 Durable**（你失能时才是最需要代理人的时候）。

## 1. Financial POA（财务委托书）

### 典型权限范围

代理人（Agent / Attorney-in-Fact）可以：
- 管理银行账户、投资账户
- 支付账单（房贷、水电、保险）
- 纳税申报
- 处理房产（买卖、出租、抵押）
- 经营生意
- 诉讼代理
- 政府福利申请（SS、Medicare）

### 两种触发方式

#### "Immediate" POA（立即生效）

- 签完立即生效
- 代理人随时可代你
- **风险**：代理人可滥用（虽不能改变 Will）
- 适合：完全信任的配偶、专业代理

#### "Springing" POA（触发式）

- 你失能后才生效
- 需医生证明 incapacity
- 更安全
- 但实践中**触发繁琐**（证明过程可能延迟 1-2 周）

### 加州 Financial POA

**执行要求**（加州 Probate Code §4121）：
- 18 岁以上、精神健全
- 书面形式
- **必须公证 OR 2 个合格见证人** 签字
- 推荐**公证**（Notarized），跨州接受度最高

**表格**：
- 加州 Uniform Statutory Form（加州通用法定表）
- 或定制化版本（律师起草）

### 代理人选择标准

- **财务能力**（理解投资、税务）
- **可靠性、诚信**
- **地理位置**（能方便处理事务）
- **时间灵活性**
- **与你的价值观一致**

### 继任代理人（Alternate）

**必须指定**：
- 首选代理人去世 / 失能 / 拒绝 → 继任接管
- 否则 POA 失效 → 需法院 Conservatorship

### 撤销

**可以随时撤销**（只要你仍有行为能力）：
- 书面撤销通知
- 销毁原 POA
- 通知所有用过的机构（银行、券商）

## 2. Healthcare POA / Advance Healthcare Directive

### 加州术语

加州把以下**合并**为一份文件 — **Advance Healthcare Directive**：
1. **Durable Power of Attorney for Health Care**（医疗代理人）
2. **Living Will**（维持生命意愿）
3. 部分 **HIPAA Release**（医疗信息共享）

### 医疗代理人（Healthcare Agent / Proxy）的权限

- 选择医生、医院、治疗方案
- 同意 / 拒绝治疗
- 访问医疗记录（HIPAA）
- 决定**临终关怀**方案
- 死后决定（尸检、器官捐献、葬礼）
- 签署"DNR"（Do Not Resuscitate）

### 加州法律要点

**生效条件**（加州 Probate Code §4700）：
- **初级医生**判定你**失去决策能力**
- 文件才生效

**执行要求**（§4673）：
- **书面**
- 签字 + **日期**
- **2 名见证人** 或 **公证**
- 见证人要求：
  - 不是医疗代理人
  - 不是医疗提供者
  - 不是受益人继承人

### 代理人选择（不同于财务）

- **价值观一致**：生死观、宗教
- **情绪稳定**：能在压力下做决定
- **地理可及**：可到场
- **愿意沟通**：跟你讨论偏好
- **能挑战医生**：不被动接受

### 常见错误

**选择"最爱你的人" ≠ 正确代理人**。最爱你的人可能**无法放手**（坚持无望治疗）。

## 3. Living Will（维持生命意愿）

### 内容

对**特定临终场景**的意愿指示：
- 终末期疾病
- 永久无意识
- 致命损伤

**典型选项**：
- 是否要**人工呼吸机**（Ventilator）
- 是否要**心肺复苏**（CPR / DNR）
- 是否要**胃管喂食**（Feeding Tube）
- 是否要**透析**
- 是否要**抗生素**
- 是否 **Hospice only**（临终关怀优先）
- 是否捐献器官
- 尸检偏好

### 与 Healthcare POA 的关系

- **Living Will** = "**做什么**"
- **Healthcare POA** = "**谁决定**"
- 两者互补

## 4. HIPAA Release（医疗信息授权）

### 为什么需要

HIPAA 法律**限制所有人**（包括配偶、成年子女）查阅你的医疗记录，除非你明确授权。

### 没有 HIPAA Release 的后果

- 配偶不能查你的检查结果
- 成年孩子不能和医生讨论你的情况
- Healthcare Agent 可能**无法及时获得信息**

### 包含在哪里

- 独立的 HIPAA Authorization Form
- 或 Advance Healthcare Directive 内嵌
- 或医疗代理人文件附件

### 授权范围

- 哪些信息可共享
- 与谁共享
- 多长时间有效
- 是否包含心理健康（额外限制）

## 5. POA 与 Trust 的关系

### Trust + POA 共同使用

**Revocable Living Trust**（见 [[Trust-类型]]）：
- 控制 Trust 内资产
- **不控制 Trust 外资产**（如没转入 Trust 的银行账户）

**Financial POA**：
- 控制 Trust **外** 资产
- 包括未 Funded 的部分
- 代理人可决定把资产转入 Trust

**两者配合使用**才能全面覆盖。

### 加州 Financial POA 典型权限

允许代理人：
- 把你的资产**转入** Revocable Trust
- 管理、出售 Trust 外资产
- 做赠与（限额）
- 处理税务

## 6. 常见陷阱

### 陷阱 1：没有 POA
失能时**唯一选择**是 Conservatorship：
- 法院程序，**6-12 个月**
- **$5,000-$20,000** 费用
- 公开记录
- 持续法院监督

### 陷阱 2：过时的 POA
老 POA 可能：
- 代理人去世 / 失能
- 与当前情况不符
- 表格格式过期（银行可能不接受）

**建议**：**每 5 年审查**。

### 陷阱 3：银行 / 券商拒绝
- 即使法定有效，有些金融机构**要求自家表格**
- 建议：**事先与银行 / 券商确认**或填写他们的 POA 补充表

### 陷阱 4：指定同一人为财务 + 医疗代理
- 可以，但**利益冲突**风险
- 医疗决定可能影响财务（如昂贵治疗）
- **分开**通常更好

### 陷阱 5：不告诉代理人
- POA 存在但代理人不知道 / 不知在哪
- **关键**：副本给代理人 + 告诉他们保管位置

## 7. 加州家庭完整包（Estate Planning Package）

**最基础的"成人文件包"**：

1. **Will**（见 [[Will-遗嘱]]）
2. **Financial POA**（Durable）
3. **Advance Healthcare Directive**（含 Healthcare POA + Living Will）
4. **HIPAA Authorization**
5. **Revocable Living Trust**（如果净资产 > $184k）
6. **Beneficiary Designations**（见 [[Beneficiary-Designation]]）

**典型费用**：
- 律师定制：$2,000-$5,000
- 在线服务（Trust & Will、FreeWill、LegalZoom）：$200-$600
- 完全 DIY：免费（但风险高）

## 8. 中美双边家庭特殊

### 国际代理问题

- 美国 POA **不自动**在中国有效
- 需要**海牙认证 / 领事认证**（依双边协议）
- 中国事务可能需要**专门的中国委托书**
- 建议：两国分别准备

### 跨境医疗

- 如你在中国就医 + 加州 Healthcare POA
- 中国医院可能不认
- 需要**中文翻译 + 当地法律认证**
- 紧急情况下**家属在场**最重要

### 继承人在中国

- 如 POA 代理人居住中国 → 签署、执行困难
- 公证可通过中国领事馆
- 但医疗场景需快速决策 → **不实用**
- **建议**：主代理人居住美国，中国亲属作为继任

## 9. 审查和更新

### 审查触发

- 搬州（州法律不同）
- 重大家庭变化（婚姻、离婚、生死）
- 代理人情况变化
- 法律变化
- **5 年定期审查**

### 执行时如何操作

你失能时家属需要：
1. **找到** 原件（告诉家人保管位置 —— 保险箱、律师保管、家庭文件柜）
2. **送给**相关机构（银行、医院）
3. **跟进**生效
