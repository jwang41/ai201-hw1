## 核心概念

**Trust（信托）= 一个法律实体，由 Trustee（受托人）为 Beneficiary（受益人）持有和管理资产**

### 三方关系

- **Settlor / Grantor / Trustor**（设立人）：放入资产的人
- **Trustee**（受托人）：管理资产的人
- **Beneficiary**（受益人）：受益于资产的人

一个人可以同时担任多种角色（如 Revocable Living Trust 中创始人 = 受托人 = 初始受益人）。

## 核心分类：Revocable vs Irrevocable

| 维度 | Revocable（可撤销） | Irrevocable（不可撤销） |
|------|-------------------|--------------------|
| 修改 | ✅ 随时 | ❌ 几乎不能 |
| 撤销 | ✅ 随时 | ❌ 不能 |
| 控制权 | 保留 | 放弃 |
| 资产所有权 | 技术上是 Trust，实际是你 | 真正转移给 Trust |
| 税务上 | 视为你本人（无区别） | 独立税务实体（某些） |
| 纳入遗产 | ✅ 是 | ❌ 否（通常）|
| 避免 Probate | ✅ | ✅ |
| 债权人保护 | ❌ | ✅ |
| 遗产税减免 | ❌ | ✅ |
| 主要用途 | Probate 规避 + 失能管理 | 税务规划 + 资产保护 |

## 1. Revocable Living Trust（可撤销生前信托）⭐ 最常见

### 核心特征

- 生前建立（"Living"）
- 可随时修改、撤销
- 你控制所有资产
- 你指定自己 + 配偶为 Trustee

### 主要好处

#### 1. 避免 Probate ⭐

这是主要原因。
- 资产在 Trust 里 → 去世后**直接按 Trust 指示** 传给受益人
- **无需法院介入**
- 节省 **9-18 个月** + **1-5% 费用**

#### 2. 失能管理（Incapacity Planning）

- 如果你失能（老年痴呆、严重伤病）
- **Successor Trustee** 自动接管
- 无需 **Conservatorship**（法院介入的监护）
- 资产继续按计划管理

#### 3. 隐私保护

- Probate 是**公开记录**（任何人可查）
- Trust 保持**私密**
- 高净值者、公众人物尤其看重

#### 4. 多州 / 跨国资产简化

- 有加州 + 德州 + 中国房产？
- Probate 需**每个州都走程序**（Ancillary Probate）
- Trust **一份文件覆盖所有**

### 缺点

- **不节税**（联邦遗产税、资本利得税都不减）
- **无债权人保护**
- **设立费用** $1,500-$5,000（律师）或 $500-$1,000（在线）
- **需要 Funding**：把资产实际转入 Trust（常被忽略）

### 关键：Funding 环节

**Trust 建立后必须把资产转入**：
- 房产：重新登记到 Trust 名下
- 银行 / 券商账户：更改账户名
- 投资账户：可能用 TOD（Transfer on Death）替代

⚠️ **未 Funded 的 Trust = 无用** —— 资产仍要走 Probate。

### Pour-Over Will 配套

- 即使 Trust 好好 Funded
- 仍需 Will 作为"**兜底**"
- 任何未转入 Trust 的资产 → 通过 Pour-Over Will 转入
- 即使没及时 Funding 也能覆盖

## 2. Irrevocable Trust（不可撤销信托）

### 种类繁多

不是一种 Trust，而是一大类：

#### 2.1 Irrevocable Life Insurance Trust (ILIT)

- 持有人寿保险保单
- 赔付**不计入遗产** → 避免 40% 遗产税
- 适合保额 > 遗产税豁免者

**示例**：高净值家庭 + $10M 终身寿险 → 不放 ILIT 则保险赔付 $10M 加上其他资产，超过 $15M 豁免 → $10M × 40% = $4M 税

#### 2.2 Charitable Remainder Trust (CRT)

- 你生前获得 Trust 收入流（5-50%）
- 死后剩余归慈善机构
- **创立当年** 获得慈善扣除
- 资本利得税递延
- 适合持有大量增值资产 + 想慈善

#### 2.3 Grantor Retained Annuity Trust (GRAT)

- 放入增值资产，获固定年金
- 超出预期回报部分给受益人
- **无需赠与税**（如结构合理）
- Walt Disney、Mark Zuckerberg 等用过

#### 2.4 Qualified Personal Residence Trust (QPRT)

- 把主要住宅转入 Trust
- 保留 N 年居住权
- N 年后房产归受益人
- **降低转移税基**（由 Trust 设立日折现）

#### 2.5 Asset Protection Trust

- 保护资产免于诉讼 / 债权人
- 需搬到**友好州**（NV、AK、DE、WY）或**离岸**（Cook Islands、Nevis）
- 医生、律师等高诉讼风险者用

#### 2.6 Special Needs Trust

- 为残疾家属设立
- 不影响政府福利资格
- 孩子有特殊需求家庭必备

#### 2.7 Spendthrift Trust

- 保护受益人免于自己挥霍
- 设定分配规则（每月固定、按里程碑、达成条件等）
- 适合**挥霍倾向**或**成瘾**的受益人

### Irrevocable 通用缺点

- **失去控制**：资产真的不是你的了
- **复杂度高**：需要持续管理、报税（Form 1041）
- **不可逆**：错了就错了
- **成本**：$3,000-$10,000+ 设立 + 持续管理费

## 3. 特殊用途 Trust

### Generation-Skipping Trust (GST)

- 跳过孩子一代，传给孙子
- 避免一次"代际"遗产税
- IRS 有 **GST 转移税** 但豁免额 $15M

### Qualified Domestic Trust (QDOT) ⭐ 跨国家庭

**美国配偶为非公民时极重要**：

- 通常配偶互相继承享**无限婚姻豁免**
- **但非公民配偶不享受这个豁免**
- QDOT = 把资产放入满足条件的 Trust → 仍可享受豁免
- 非公民配偶按一般分配获得资产时缴税

**适合**：
- 美国公民 + 中国国籍（或其他非公民）配偶
- 高净值家庭（> $15M 豁免）

### Dynasty Trust

- 跨多代持续
- 主要目的：**代际财富保护**
- 资产**永不纳入**下代遗产
- 少数州允许"永久"（如南达科他、德拉瓦）

### Credit Shelter Trust / Bypass Trust

- 历史上用来"利用双方豁免额"
- 2013 年起 **Portability** 规则让这个 Trust 不那么必要
- 但**州遗产税** 仍可能用到

## 4. 高净值家庭典型架构

对于净资产 **$2M-$15M**（标准美国富裕家庭）：

```
基础层：
  ├── Revocable Living Trust（避 Probate + 失能管理）
  ├── Pour-Over Will（兜底）
  ├── Financial POA
  ├── Advance Healthcare Directive
  ├── HIPAA Release
  └── Beneficiary Designations（401K/IRA/Life Insurance）

增强层（如需）：
  ├── ILIT（大额人寿保险）
  ├── Asset Protection（医生/律师）
  ├── Special Needs Trust（残疾家属）
  └── QDOT（非公民配偶）
```

净资产 **$15M+**：
- 加上 GRAT、CRT、QPRT、Dynasty Trust
- 需要专业团队（律师 + CPA + Financial Advisor）

## 5. Trust 的税务处理

### Revocable Trust

**视同设立人本人**：
- 没有单独税号
- 所有收入 / 支出在你 1040 报
- 无额外报税

### Irrevocable Trust

**独立税务实体**：
- 自己的 EIN
- 每年填 **Form 1041**
- **极高税率**：2026 年 Trust 37% 税率门槛仅 **$15,650**（对比个人 $640k+）
- 通常分配给受益人以避免 Trust 税率

## 6. 加州特殊

### Proposition 19（2021 起生效）

影响房产转入 Trust：
- 加州 Prop 13 **低房产税基** 的继承规则变严
- 只有**父母 → 子女** + 子女**将房产作为主要住宅** 可继承低税基
- 其他情况下 → 按**当前市值**重估
- 继承房产但不居住 → 税基大幅上升

**策略启示**：Trust 中房产继承需重新考虑（可能税务上不再省钱）。

### 加州的 Revocable Trust 流行

- 加州 Probate 费用特别高
- **>90% 高净值家庭**用 Revocable Trust
- 律师费仅 $2-5k 就能 setup → 节省 $50k+ Probate

## 7. 中美双边家庭

### 跨国 Trust 复杂度

- 美国 Trust 的**中国资产**：可能**不被中国法律承认**
- 中国资产需**另外规划**（可能用 Will 或不动产直接继承）
- 税务：**FATCA / FBAR** 规则持续适用

### 非公民配偶

- **QDOT 几乎必需**（见上）
- 或考虑**赠与计划**在生前转移资产

### 放弃身份规划

- 如计划放弃绿卡 → 先把资产放入 Trust 还是放弃后？
- 放弃后**Section 2801** 对美国受益人 40% 税
- 复杂规划，需专业咨询
